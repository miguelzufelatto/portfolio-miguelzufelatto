# 💻 Meu Portfólio Pessoal

Este repositório contém o código-fonte da minha landing page pessoal. O objetivo deste projeto é centralizar minha trajetória, projetos e habilidades em um único lugar, servindo como meu cartão de visitas para recrutadores e parceiros de projetos.

🔗 **Acesse o site:** [https://miguelzufelatto.github.io/portfolio-miguelzufelatto/](https://miguelzufelatto.github.io/portfolio-miguelzufelatto/)

## 🚀 Sobre o Projeto

O design foi construído do zero com foco em minimalismo e eficiência técnica. A estética segue uma linha industrial/terminal, utilizando fontes monoespaçadas e um esquema de cores escuro para refletir minha identidade como Engenheiro de Software e Analista de Dados.

### 🛠️ Tecnologias Utilizadas
- **HTML5** e **CSS3** (Estrutura e Estilização personalizada)
- **JavaScript** (Manipulação do DOM e interatividade)
- **Google Fonts** (Tipografia monoespaçada estratégica)

## 🎨 Design System

O projeto segue um guia de estilo rigoroso para garantir consistência:
- **Cores:** Fundo `#0b0b09`, texto `#eeeeec`, destaque `#868469`.
- **Tipografia:** Foco total em fontes monoespaçadas (ex: JetBrains Mono ou Fira Code).
- **Layout:** Estrutura mobile-first, centralizada no Hero e alinhada à esquerda nas seções de leitura.

## 📁 Estrutura de Seções
1. **Hero:** Apresentação direta com CTA para projetos.
2. **Sobre:** Narrativa pessoal sobre minha jornada e motivações.
3. **Habilidades:** Exibição visual de ferramentas (sem barras de progresso).
4. **Projetos:** Cards detalhados com links e tecnologias.
5. **Contato:** Links diretos para comunicação.
